# be-the-hero
## Application to register NGOs, their cases and needs. so that people can contribute.

> Status: completed

## Techs

+ Node.js
+ SQLite
+ ReactJS
+ React Native
+ Expo
+ Styled-Components


## How to use

+ Run ```npm start``` in the folder backend
+ Run ``` npm start``` in the folder frontend
